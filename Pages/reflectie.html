Reflectie

Hier staat hoe ik deze opdracht aangepakt heb.
Ik vertel bijvoorbeeld dingen over het ontwerp, hoe ik de site heb gemaakt en hoe sommige dingen precies werken.

Hoe werkt een website?

Bij het maken van een website heb je meerdere mogelijkheden, zo heb je bijvoorbeeld website bouwers waarmee je makkelijk websites kan maken zonder dat je veel kennis ervan hebt. Ook kan je zelf de codering van websites schrijven, dit is ook wat ik doe. Hierbij moet je gebruik maken van meerdere programmeer talen, namelijk HTMl, CSS en JavaScript.

HTML

HTML is de taal die ervoor zorgt dat alle objecten zoals bijvoorbeeld text te zien zijn, dit is dus de basis van de website.

CSS

CSS gebruik je om de objecten een ander uiterlijk te geven, zo kan je hiermee kleuren, grootte, plaats en nog veel meer aanpassen.
In deze site is dit onder andere gebruikt voor de kleur en plek van de navigatie balk, achtergronden en alle andere dingen die te zien zijn.

JavaScript

JavaScript zorgt ervoor dat er bepaalde acties gebeuren als je bijvoorbeeld op een knopje klikt, zo heb ik dit bijvoorbeeld gebruikt voor de pagina overgangenvan.

Het combineren

Het eerste wat je gebruikt bij het maken van een website is HTML, je zet dingen neer in de site zodat je ze daarna met CSS een ander uiterlijk kan geven.
De HTML en CSS word meestal ene beetje om en om gebruikt, dus dat ik eerst bijvoorbeeld dingen in HTML neerzet, vervolgens met de CSS aan de slag ga en daarna weer met HTML.


Ontwerp

Voor het ontwerp heb ik veel op internet gekeken voor inspiratie.
Ik ben bijvoorbeeld een effect tegen gekomen genaamd parallax, dit heb ik gebruikt voor de home pagina. Wat dit effext doet en hoe het precies werkt leg ik later uit. Ook wilde ik graag de naam altijd aan de bovenkant van de website hebben staan, dan is het namelijk goed te zien van welk bedrijf de website is.

Het maken van de site

Ik ben begonnen met het parallax effect op de home pagina, toen ik uitgevonden had hoe dit moest en ik had het werkend ben ik aan de slag gegaan met de kaartjes eronder.
Daarna kwam ik op het idee om pagina overgangen voor de pagina's, hiervoor heb ik een paar dingen nog moeten aanpassen zodat alleen de inhoud van de pagina veranderd en bijvoorbeeld niet de navigatie balk. Om deze overgangen werkend te krijgen was nog het moeilijkste, het werkte namelijk wel, maar de achtergrond van de home pagina verschoof steeds en het witte vlak verscheen in een keer in plaats van met een animatie.

Hoe werken dingen precies?

Parallax

Het parallax effext heb ik gebruikt op de home pagina zoals hier rechts te zien is.
Dit is gemaakt door meerdere lagen achter elkaar te plaatsten, de achterste laag is wat verder naar achter, hierdoor gaat deze minder snel naar beneden bij het scrollen. Deze website laat goed zien hoe dit er uit ziet. (Klik na het openen op links boven)

Pagina overgangen

Voor de overgangen van de pagina's heb ik ervoor gezorgd dat er met een animatie eerst een wit vlak voor de pagina verschijnt, daarna wordt de inhoud van de pagina veranderd en vervolgens verdwijnt het vlak weer.
