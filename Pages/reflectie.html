Reflectie

Hier staat hoe ik deze opdracht aangepakt heb.
Ik vertel bijvoorbeeld dingen over het ontwerp, hoe ik de site heb gemaakt en hoe sommige dingen precies werken.

Hoe werkt een website?

Ontwerp

Voor het ontwerp heb ik veel op internet gekeken voor inspiratie.
Ik ben bijvoorbeeld een effect tegen gekomen genaamd parallax, dit heb ik gebruikt voor de home pagina. Wat dit effext doet en hoe het precies werkt leg ik later uit. Ook wilde ik graag de naam altijd aan de bovenkant van de website hebben staan, dan is het namelijk goed te zien van welk bedrijf de website is.

Hoe werken dingen precies?

Parallax

Het parallax effext heb ik gebruikt op de home pagina zoals hier rechts te zien is.
Dit is gemaakt door meerdere lagen achter elkaar te plaatsten, de achterste laag is wat verder naar achter, hierdoor gaat deze minder snel naar beneden bij het scrollen. Deze website laat goed zien hoe dit er uit ziet. (Klik na het openen op links boven)
